# Linux

	sudo apt-get install git python cc65
	pip install pillow
	git clone https://github.com/zohassadar/NestrisHacks
	cd NestrisHacks

To build `tetris.nes`:

	bash build.sh


# OS X


In **Terminal**, run:
	
	brew install cc65
	pip install pillow
	git clone  https://github.com/zohassadar/NestrisHacks
	cd NestrisHacks
	
To build `tetris.nes`:

	bash build.sh


# Windows

These steps are untested. 

Use Cygwin and the version of cc65, or use the Windows Subsystem for Linux (WSL)
